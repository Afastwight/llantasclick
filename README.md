# LlantasClick

Proyecto web tipo landing page para la venta de llantas online en Ecuador.  
La página presenta catálogo de productos, categorías, marcas, búsqueda por compatibilidad, reseñas, ventajas del servicio y contacto por WhatsApp.

## Vista general

LlantasClick es una página web estática desarrollada con HTML, CSS y JavaScript.  
El objetivo del proyecto es ofrecer una interfaz moderna para que los usuarios puedan consultar llantas disponibles según el tipo de vehículo, revisar productos destacados y solicitar información o cotización.

## Características principales

- Página principal con diseño moderno y responsivo.
- Catálogo de llantas destacadas.
- Filtro de productos por categoría:
  - Sedán
  - SUV
  - Comercial
  - Sport
- Sección de marcas disponibles.
- Herramienta visual para búsqueda de compatibilidad.
- Sección de ventajas del servicio.
- Botón flotante de WhatsApp.
- Animaciones al hacer scroll.
- Estructura organizada por carpetas.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Vite
- Vercel
