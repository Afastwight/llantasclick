Aviso de Protección de Datos Personales

LlantasClick informa a los usuarios que los datos personales proporcionados serán tratados de forma responsable, segura y únicamente para fines relacionados con la prestación del servicio.

Responsable del tratamiento

El responsable del tratamiento de datos será LlantasClick o la persona natural/jurídica que administre oficialmente la plataforma.

Datos tratados

Se podrán tratar datos identificativos, datos de contacto, información de pedidos, preferencias de compra y datos necesarios para atención al cliente.

Finalidades

Los datos se usarán para gestionar compras, procesar pagos, emitir comprobantes, coordinar entregas, brindar soporte y cumplir obligaciones legales.

Seguridad

Se aplicarán controles de acceso, autenticación segura, respaldo de información y medidas para prevenir accesos no autorizados.

Derechos del titular

El titular podrá solicitar acceso, rectificación, eliminación, oposición o actualización de sus datos conforme a la normativa aplicable.
