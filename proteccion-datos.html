Aspecto legal

Aviso de Protección de Datos Personales

LlantasClick informa a los usuarios que los datos personales proporcionados a través del sitio web serán tratados de forma responsable, segura y únicamente para fines relacionados con la prestación del servicio.

Responsable del tratamiento

El responsable del tratamiento de datos será LlantasClick o la persona natural o jurídica que administre oficialmente la plataforma y sus servicios comerciales.

Datos personales tratados

Se podrán tratar datos identificativos, datos de contacto, información de pedidos, historial de compras, preferencias de productos, datos de facturación y datos necesarios para brindar atención al cliente.

Finalidades del tratamiento

Los datos se utilizarán para gestionar compras, procesar pagos, emitir comprobantes, coordinar entregas, responder solicitudes, brindar soporte, administrar cuentas de usuario y cumplir obligaciones legales.

Base de seguridad

LlantasClick aplicará medidas de seguridad como control de acceso, autenticación de usuarios, gestión segura de contraseñas, protección de sesiones, respaldo de información y restricción de acceso a datos sensibles.

Autenticación de usuario

Para acceder a funciones como perfil de usuario o historial de pedidos, el usuario podrá utilizar un sistema de autenticación mediante correo electrónico y contraseña. Las credenciales deberán ser gestionadas de forma segura.

Gestión segura de contraseñas

Las contraseñas no deben compartirse con terceros. El usuario es responsable de mantener la confidencialidad de sus datos de acceso. En caso de olvido o sospecha de acceso no autorizado, podrá utilizar el proceso de recuperación de contraseña.

Copias de seguridad

La plataforma podrá aplicar mecanismos de respaldo de información para reducir riesgos de pérdida de datos, mejorar la continuidad del servicio y proteger la información de pedidos, usuarios y operaciones.

Derechos del titular

El titular de los datos podrá solicitar acceso, rectificación, actualización, eliminación u oposición al tratamiento de sus datos personales, conforme a la normativa aplicable.

Confidencialidad

La información personal será utilizada únicamente para los fines informados y no será vendida ni compartida con terceros no autorizados, salvo obligación legal o necesidad operativa del servicio.
