Política de Privacidad

En LlantasClick respetamos la privacidad de nuestros usuarios y nos comprometemos a proteger la información personal proporcionada durante el uso del sitio web.

Datos que recopilamos

Podemos recopilar datos como nombre, correo electrónico, teléfono, dirección, información de pedidos y datos necesarios para procesar compras o cotizaciones.

Finalidad del tratamiento

Los datos se utilizan para gestionar pedidos, responder consultas, procesar pagos, mejorar el servicio y cumplir obligaciones legales.

Protección de datos

Se aplican medidas técnicas y organizativas para proteger la información contra accesos no autorizados, pérdida, alteración o divulgación indebida.

Derechos del usuario

El usuario puede solicitar acceso, actualización, rectificación o eliminación de sus datos personales mediante los canales de contacto del sitio.
